# SCT_DA_4
## Customer Personality Analysis Dashboard using Tableau Public | SkillCraft Data Analytics Internship Task 4
## Objective 
Perform Exploratory Data Analysis (EDA) on a marketing campaign dataset and generate business insights using Tableau Public
## Tools used
- tableau Public
- Microsoft Excel
- Kaggle Dataset
## Dashboard Insights
- Customer Count by Education
- Customer Count by Marital Status
- Average Income by Education
- Product Spending Analysis
## Key Findings
- Graduates form the largest customer segment.
- Married customers represent the largest customer group.
- PhD and master degree holders have higher average income.
- Wine products generate the highest spending among customers.
